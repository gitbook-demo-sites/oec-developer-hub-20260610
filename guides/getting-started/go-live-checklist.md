# Go-live checklist

- Sandbox validation complete for the required transaction set
- Error handling covers retryable and terminal failures
- Webhook receiver is resilient and observable
- Production credentials issued and verified
- Support handoff path confirmed
- Internal OEC rollout notes reviewed by the launch owner

{% hint style="warning" %}
If launch-critical internal notes exist, they should live in authenticated portal overlays, not email threads or ad hoc docs.
{% endhint %}
